# Geospatial Data Science exercises

Each folder herein contains the exercises and course material for each session. If you would like to create a stand along environment, follow these steps:

```
C:\path_to_repo\ conda create --name geo_test geopandas rasterio
C:\path_to_repo\ conda activate geo_test
C:\path_to_repp\ pip install -r exercies\requirements.txt
```

1. The following commands are tailored to a Windows machine inside the World Bank.  
2. On windows, pip will not properly install GDAL, but conda will.
